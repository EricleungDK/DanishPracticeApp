## Technical Decisions
- **Capacitor over React Native**: follows OCDReliever pattern, 100% UI code reuse via WebView instead of rewriting all components
- **API abstraction layer**: `AppAPI` interface + `ApiProvider` context + `getApiInstance()` singleton decouples stores from Electron's `window.api`, enabling same code on mobile
- **Separate synonym content**: dedicated `synonyms.json` + DB table rather than deriving from wordlist English overlaps, ensures quality synonym pairs
